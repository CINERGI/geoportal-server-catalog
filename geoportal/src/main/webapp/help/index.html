How to Use the Data Discovery Studio

Searching

Enter a term or a phrase to search all indexed fields (titles, abstracts, keywords, organizations, etc.) For multiple words “and” is assumed (i.e. coral bleaching will search for records containing coral and bleaching. You may also use OR to find records with either term, or NOT to exclude terms. Search terms are italicized in the returned records.

This shows the current search filters. Click the X to quickly remove a filter from the search.

To search geospatially using the map, zoom to the area of interest, and then select within (all points in a dataset are inside the map extent) or intersects (some points within a dataset are inside the map extent). Alternately, you can click on “Bounding Box” to enter latitude and longitude bounds to search.

Narrow your search by specifying additional facets; click on a facet name to see search tools, sub-categories, or pick lists for that facet.  Facets in bold were automatically constructed by the metadata enhancement pipeline behind the DDH system.

The central pane holds resources matching your search results. Which brings us to...

Working with Search Results

Actions to take with an individual record. Not all will be available for all records

  • HTML shows the record’s metadata in a human-readable form, including metadata elements automatically added by the CINERGI pipeline that underlies DDH. This information should help you decide if the dataset fits for your purpose.
  • ISO XML and JSON present the metadata in alternate (machine readable) formats.
  • Get Data lists URLs found in the metadata record. Depending on the type of data, they may point to a downloadable dataset, an online portal, a web service endpoint, respective documentation, etc
  • Thumbnail gives a larger image of the map view on the right.
  • FTP gives an FTP download site, if available
  • Studio  allows you to pass the metadata record to a collection of notebooks served from a Jupyter Hub. Using Jupyter notebooks one can download and visualize the data, conduct analysis, and execute models. The collection of notebooks, as well as registered JupyterHub servers, is growing – please explore it!
  • Provenance shows the steps taken to enhance the metadata once ingested at DDS
  • Edit launches the metadata editor tool. See something wrong with the metadata? you can help improve the DDS metadata! See the Help information on the metadata editor tool for more.
  • Schema.org provides machine readable metadata for search engines.
